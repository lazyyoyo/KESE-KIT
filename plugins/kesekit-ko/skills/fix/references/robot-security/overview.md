# 로봇 보안 개요

> 출처: 한국인터넷진흥원(KISA) 「로봇 보안모델(고도화)」 + 「로봇 보안취약점 점검 체크리스트 해설서」

## 1. 개요

| 항목 | 내용 |
|------|------|
| 발행 | 한국인터넷진흥원(KISA) |
| 대상 | 로봇 시스템 제조사 및 운영자 |
| 범위 | 산업용/서비스용/의료용 로봇 |
| 라이프사이클 | 설계 → 개발 → 배포 → 운용 → 폐기 |

## 2. 로봇 유형 분류 (ISO 8373)

| 유형 | 설명 | 보안 특성 | 사례 |
|------|------|----------|------|
| 산업용 | 제조 공정 자동화, 생산 라인 | 제어 시스템 안정성·안전성 | 용접/도장, 팔레타이징, 휴머노이드 |
| 서비스용 | 물류, 청소, 안내, 경비 | 이동성·외부 시스템 연동 | AMR, 안내로봇, 경비로봇, 재난대응 |
| 의료용 | 수술 보조, 재활, 환자 케어 | 데이터 무결성·신뢰성 | 수술로봇, 병원 물류, 재활로봇 |

## 3. 로봇 시스템 구성요소

| 구성요소 | 세부 내용 |
|---------|----------|
| 로봇 본체 | 제어부, 센서부, 구동부, 시스템/응용 SW |
| 지원 인프라 | 도킹/충전 스테이션, 네트워크 장비, 전력 |
| 사용자 인터페이스 | HMI/조작 콘솔, 모바일 앱, 웹 대시보드 |
| 운영 관리 시스템 | 연동 서버, DB, 클라우드, 원격 관리 |

## 4. 로봇 속성 분류 (MITRE EMB3D 기반)

| 속성 | 세부 |
|------|------|
| 하드웨어 | 마이크로프로세서, 외부/내부 인터페이스, 메모리/저장장치 |
| 네트워크 | 산업용 이더넷, 필드버스, Wi-Fi, 5G, MQTT, ROS/ROS2 DDS |
| 시스템 SW | RTOS, 펌웨어, 디바이스 드라이버 |
| 응용 SW | 로봇 제어 앱, HMI, 관제 시스템, 클라우드 서비스 |

## 5. 로봇 보안 목표 (6가지)

| 목표 | 설명 |
|------|------|
| 기밀성 | 로봇 데이터·통신의 비인가 접근 방지 |
| 무결성 | 제어 명령·데이터의 무단 변조 방지 |
| 가용성 | 로봇 서비스의 지속적 운영 보장 |
| 정확성 | 로봇 동작·판단의 정밀성 보장 |
| 신뢰성 | 로봇 시스템의 일관된 동작 보장 |
| 안전성 | 인명·재산 피해 방지 (로봇 특화) |
