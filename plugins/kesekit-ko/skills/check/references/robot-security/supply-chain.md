# 공급망 보안 체크리스트

> NIST SP 800-161 기반 로봇 시스템 공급망 보안 요구사항

## 체크리스트 항목 (7개)

| 코드 | 항목 | 참조 표준 |
|------|------|----------|
| SC-01 | 오픈소스 소프트웨어 구성요소 관리 | CRA Part 2.1/2.6, IEC 62443-4-1 SM-9 |
| SC-02 | SBOM(소프트웨어 자재 명세서) 관리 | CRA Part 2.1, NIST SP 800-161 |
| SC-03 | 공급망 위험 평가 | NIST SP 800-161, NIS2 |
| SC-04 | 서드파티 컴포넌트 보안 검증 | NIST SP 800-161 RA-9 |
| SC-05 | 공급망 무결성 모니터링 | NIST SP 800-161 SI-2/4/5 |
| SC-06 | 공급자 보안 요구사항 관리 | NIST SP 800-161 SI-7 |
| SC-07 | 공급망 사고 대응 체계 | NIST SP 800-161 SI-2 |

### 핵심 요구사항
- Log4j, SolarWinds 등 공급망 공격 사례 대응
- SBOM 생성·관리·갱신 체계 구축
- 오픈소스 취약점 자동 스캔 (Dependabot, Snyk 등)
- 서드파티 컴포넌트 보안 평가 프로세스
