### NIST SP 800-218 실천 영역

| 영역 | 설명 |
|------|------|
| PO (Prepare Organization) | 조직이 안전한 SW 개발을 수행할 수 있도록 준비 |
| PS (Protect Software) | SW를 변조 및 무단 접근으로부터 보호 |
| PW (Produce Well-Secured SW) | 보안 취약점을 최소화한 SW 개발 |
| RV (Respond to Vulnerabilities) | 배포된 SW 취약점 식별 및 대응 |
