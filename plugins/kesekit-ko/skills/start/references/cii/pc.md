### 주요 점검 명령어
```powershell
# PC-01: 비밀번호 최대 사용기간
net accounts | findstr "최대"

# PC-04: 공유 폴더
net share

# PC-13~14: 백신
Get-MpComputerStatus

# PC-15: 방화벽
Get-NetFirewallProfile | Select Name, Enabled

# PC-16: 화면보호기
reg query "HKCU\Control Panel\Desktop" /v ScreenSaveTimeOut
```
