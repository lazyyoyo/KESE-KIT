# KESE - KISA Enhanced Security Evaluation Kit

प्रमुख सूचना एवं संचार अवसंरचना (CII) की कमजोरी विश्लेषण-मूल्यांकन तथा AI सुरक्षा मूल्यांकन के लिए Claude Code स्किल प्लगइन।

---

## अवलोकन

KESE (KISA Enhanced Security Evaluation Kit) एक Claude Code प्लगइन है जो KISA (कोरिया इंटरनेट एवं सुरक्षा एजेंसी) दिशानिर्देशों पर आधारित व्यापक सुरक्षा कमजोरी मूल्यांकन क्षमताएँ प्रदान करता है। यह प्रमुख सूचना अवसंरचना (CII) और AI सुरक्षा दोनों प्रकार के मूल्यांकनों का समर्थन करता है।

## सुविधाएँ

| स्किल | विवरण |
|-------|-------|
| `/kesekit-en:start` | पूर्ण सुरक्षा कमजोरी मूल्यांकन चलाएँ (CII 560+ आइटम या AI सुरक्षा) |
| `/kesekit-en:check` | तैनाती-पूर्व सुरक्षा अनुपालन चेकलिस्ट |
| `/kesekit-en:fix` | हार्डनिंग स्क्रिप्ट और सुरक्षा सुधार स्वचालित रूप से तैयार करें |
| `/kesekit-en:guide` | AI उपकरणों के लिए सुरक्षित कोडिंग प्रॉम्प्ट तैयार करें |

## समर्थित दिशानिर्देश

### 1. CII (प्रमुख सूचना अवसंरचना) — 560+ आइटम

**तकनीकी मूल्यांकन**
| सिस्टम | कोड | आइटम |
|--------|------|:-----:|
| Unix/Linux सर्वर | U-01~U-67 | 67 |
| Windows सर्वर | W-01~W-64 | 64 |
| वेब सेवा | WEB-01~WEB-26 | 26 |
| सुरक्षा उपकरण | S-01~S-23 | 23 |
| नेटवर्क उपकरण | N-01~N-38 | 38 |
| नियंत्रण प्रणाली | C-01~C-51 | 46 |
| PC | PC-01~PC-18 | 18 |
| DBMS | D-01~D-26 | 26 |
| मोबाइल | M-01~M-04 | 4 |
| Web Application | 21 कोड | 21 |
| वर्चुअलाइज़ेशन | HV-01~HV-25 | 25 |
| क्लाउड | CA-01~CA-19 | 19 |

**प्रशासनिक मूल्यांकन**: A-1~A-127 (127 आइटम, 14 क्षेत्र)
**भौतिक मूल्यांकन**: P-1~P-18 (18 आइटम)

### 2. AI सुरक्षा गाइड — 54+ आइटम

| लक्षित समूह | आइटम | जीवनचक्र |
|-------------|:-----:|----------|
| AI डेवलपर | 54 | 6-चरण (योजना→डेटा→मॉडल→तैनाती→निगरानी→विलोपन) |
| सेवा प्रदाता | ~43 | 6-चरण (योजना→विकास→संचालन→रखरखाव→प्रतिक्रिया→विलोपन) |
| उपयोगकर्ता | 7 | सुरक्षा सर्वोत्तम प्रथाएँ |

**संरचना पुनर्गठन — Progressive Disclosure पैटर्न लागू**

| ज्ञान संग्रहण | SKILL.md में हार्डकोडेड | `references/` में पृथक (18 फ़ाइलें) |
| आइटम कोड | केवल कुछ आइटम शामिल | 2026 गाइड आधारित सभी आइटम |
| विस्तारशीलता | नए दिशानिर्देश जोड़ने पर स्किल संख्या बढ़ती | स्किल 4 स्थिर, केवल references जोड़ें |

**नया दिशानिर्देश जोड़ा गया: AI सुरक्षा गाइड**
- स्रोत: विज्ञान एवं ICT मंत्रालय · KISA 「कृत्रिम बुद्धिमत्ता (AI) सुरक्षा गाइड」
- AI डेवलपर 54 सत्यापन आइटम (6-चरणीय जीवनचक्र)
- AI सेवा प्रदाता सुरक्षा आवश्यकताएँ
- AI उपयोगकर्ता सुरक्षा नियम 7

**CII दिशानिर्देश अपडेट**
- 2026 विस्तृत गाइड के आधार पर सभी आइटम पुनः निकाले गए
- आइटम कोड प्रणाली परिलक्षित (WEB, HV, CA आदि नए कोड)

### v1.0.0 (2026-03-29)
